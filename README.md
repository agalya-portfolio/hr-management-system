# HR HUB CONNECT : Dynamic HR System with Real-Time Tracking

## Project Overview

HR HUB CONNECT is a web-based Human Resource Management System developed to automate and simplify HR operations within an organization. The system provides a centralized platform for managing employee records, attendance, leave tracking, salary details, project assignments, and performance monitoring with real-time analytical insights.

The project focuses on improving efficiency, reducing manual paperwork, and providing interactive dashboards for faster HR decision-making through data visualization.

---

## Objectives

- Automate HR-related operations
- Manage employee information efficiently
- Track attendance and leave records
- Monitor employee performance
- Generate real-time analytical reports
- Provide a responsive and user-friendly interface

---

## Project Workflow

1. Admin logs into the system
2. Employee data is entered and managed
3. Data is stored in MySQL database
4. PHP retrieves and processes data
5. Chart.js generates real-time visual reports
6. Dashboard updates dynamically with new records
